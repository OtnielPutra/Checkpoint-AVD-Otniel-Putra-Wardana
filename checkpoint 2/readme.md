# Data Understanding

pada bagian data understanding, kita diminta untuk memahami dari data yang kita miliki. seperti
* Struktur Data (Informasi Lanjutan)
* Statistik Deskriptif
* Tipe Data Setiap kolom
* Inconsistent Values
* Missing Values
* Duplicated Values
* Check Outliers

Data Understanding digunakan untuk memahami isi, struktur, dan kualitas data sebelum dilakukan analisis lebih lanjut. Tahap ini bertujuan mengidentifikasi tipe data, pola distribusi, nilai yang hilang (missing values), duplikasi, inkonsistensi, serta kemungkinan outlier agar data yang digunakan benar-benar akurat dan relevan. Dengan memahami karakteristik data sejak awal, proses analisis menjadi lebih terarah, kesalahan dapat diminimalkan, dan hasil yang diperoleh menjadi lebih valid serta dapat dipercaya.

# Eksplorasi Data (EDA)

Eksplorasi Data (EDA) digunakan untuk menganalisis dan memahami pola, hubungan, serta karakteristik utama dalam dataset melalui statistik deskriptif dan visualisasi seperti histogram, grafik, atau diagram. Tahap ini membantu menemukan tren, anomali, distribusi data, serta hubungan antar variabel sehingga analisis menjadi lebih terarah dan berbasis fakta. 

**Insight** sebagai bagian dari EDA merupakan hasil pemahaman yang diperoleh dari proses eksplorasi tersebut, berupa kesimpulan atau temuan penting yang dapat menjawab permasalahan, mendukung pengambilan keputusan, dan menjadi dasar rekomendasi yang lebih strategis.
